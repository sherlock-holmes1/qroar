# Product Overview

**qroar** is a Chrome extension that provides 1-click QR code generation for marketers and users who need quick QR codes.

## Core Features
- Generate QR codes instantly for the current page URL
- Customizable QR code designs with presets (Blue, Red, etc.)
- Options page for advanced customization
- Support for both Chrome and Firefox browsers

## Target Users
- Marketers who need quick QR code generation
- Users who frequently share URLs via QR codes
- Anyone needing fast, customizable QR code creation

## Key Value Proposition
The extension eliminates the friction of QR code generation by providing instant, one-click creation with professional design options, making it ideal for marketing workflows and quick sharing scenarios.
